# pong-raylib
Pong game in C using Raylib library for graphics

# Requirements
<ul>
  <li><a href="https://www.raylib.com">Raylib</a></li>
</ul>

# How to run
```console
$ ./build.sh
```

# How to play
Player in left move with `Z` to up and `X` to down

Player in right move with `key up` and `key down`
